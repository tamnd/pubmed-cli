First release of pubmed-cli.

Search PubMed biomedical literature from the command line. Reads the NCBI
E-utilities API -- no API key required.

## Commands

- `pubmed search <query>` -- search PubMed, returns article list
- `pubmed article <pmid>` -- show article metadata for a PMID
- `pubmed abstract <pmid>` -- show full abstract text
- `pubmed related <pmid>` -- find related articles via NCBI link scoring

## Output formats

All commands support `-o table|json|jsonl|csv|tsv|url|raw` and pipe-friendly
defaults (table on TTY, JSONL when piped).
